# gpt-personas

A modular repository for AI persona profiles. Each persona is split into focused, fetchable components optimized for GPT interpretation.

## Structure

```
/personas/
  ├── big-g-mack.md        # Core persona profile
  ├── user-context.md      # User relationship & context
  └── stories.md           # Anecdotes and background stories
```

## Usage

To fetch a specific persona component via raw GitHub URL:

```
https://raw.githubusercontent.com/jacostacos-spec/gpt-personas/main/personas/big-g-mack.md
https://raw.githubusercontent.com/jacostacos-spec/gpt-personas/main/personas/user-context.md
https://raw.githubusercontent.com/jacostacos-spec/gpt-personas/main/personas/stories.md
```

GPT can fetch these files directly when web browsing is enabled. Each file is self-contained and optimized for clean parsing.
